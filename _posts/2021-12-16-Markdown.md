## 0. Markdown이란?
- 마크업 언어의 일종

- MS Word와 같은 문서 작성 프로그램 없이 텍스트만으로 여러 서식을 가진 문서를 작성하는 방법

- <파일명>.md으로 파일을 생성해 사용 (<파일명>.markdown도 가능)<br/><br/><br/>



## 1. 제목
제목은 #을 이용해 작성할 수 있다. 

#은 1개부터 6개까지 사용할 수 있고, #의 숫자가 많아질수록 크기가 작아진다.

### **■ Markdown**  
```
# Title 1

## Title 2

### Title 3

#### Title 4

##### Title 5

###### Title 6
```

### **■ Result**  
# Title 1

## Title 2

### Title 3

#### Title 4

##### Title 5

###### Title 6<br/><br/><br/>



## 2. 글꼴 스타일
기본적으로 기울임, 볼드, 취소선을 사용할 수 있다.

### **■ Markdown**  
```
기울임 : *기울임* _기울임_

볼드 : **볼드** __볼드__

취소선 : ~~취소선~~
```

### **■ Result**  
기울임 : *기울임* _기울임_ 

볼드 : **볼드** __볼드__

취소선 : ~~취소선~~<br/><br/>



## 3. 줄바꿈
평소처럼 Enter를 사용해 줄을 바꾸면 줄바꿈이 되지 않는다.

문장을 다음 줄에 쓰고 싶다면 : 문장 맨 끝에 공백(띄어쓰기) 두 칸을 입력  

문단을 바꾸고 싶다면 : 빈 줄 입력  

아무것도 작성하지 않은 빈 줄을 입력하고 싶다면 : `<br/>` 삽입  

※ Enter를 이용한 빈 줄은 여러 개 입력해도 문단만 바뀌고 빈 줄이 삽입되지 않는다.

### **■ Markdown**  
```
문장 끝에 공백 두 칸을 입력합니다.  
줄이 바뀌었습니다.


빈 줄을 입력해 문단을 바꿉니다.


빈 줄을 입력합니다.<br/><br/><br/>
빈 줄이 삽입되었습니다.
```

### **■ Result**  
문장 끝에 공백 두 칸을 입력합니다.  
줄이 바뀌었습니다.


빈 줄을 입력해 문단을 바꿉니다.


빈 줄을 입력합니다.<br/><br/><br/>
빈 줄이 삽입되었습니다.<br/><br/><br/>



## 4. 인덱스
`1. 2. 3. `과 같이 숫자와 .으로 순서가 있는 인덱스를 만든다.

`- + *`으로 순서가 없는 인덱스를 만든다. (- + * 모양과 관계없이 인덱스는 모두 동일한 모양)

tab 키나 공백 4칸으로 하위 항목을 만들 수 있다.

순서가 없는 리스트는 하위 항목을 만들면 상위 항목과 다른 모양의 인덱스가 생성된다.

### **■ Markdown**  
```
1. 1번 인덱스

2. 2번 인덱스

    1. 2-1번 인덱스

    2. 2-2번 인덱스
```
```
* 순서가 없는 인덱스

* 순서가 없는 인덱스

    - 순서가 없는 인덱스

    - 순서가 없는 인덱스

        + 순서가 없는 인덱스

        + 순서가 없는 인덱스
```

### **■ Result**  
1. 1번 인덱스

2. 2번 인덱스

    1. 2-1번 인덱스

    2. 2-2번 인덱스<br/><br/>

* 순서가 없는 인덱스

* 순서가 없는 인덱스

    - 순서가 없는 인덱스

    - 순서가 없는 인덱스

        + 순서가 없는 인덱스

        + 순서가 없는 인덱스<br/><br/><br/>



## 5. 코드 블럭
`(백틱)을 이용해 코드 블럭을 만든다.

문장 중간에 인라인 블럭을 작성하려면 ` 하나로 코드를 감싼다.

코드 여러 줄을 블럭으로 감싸려면 코드 위아래에 ```을 삽입한다.

위 쪽 ``` 옆에 코드의 언어를 입력해주면 코드가 하이라이트 된다.

### **■ Markdown**  
    문장 중간에 `인라인 코드`를 삽입합니다.

    ```python
    nums = [6, 1, 7, 23, 10, 24]
    total = 0
    for i in nums:
        total += i
    print(total)
    ```

### **■ Result**  
문장 중간에 `인라인 코드`를 삽입합니다.<br/><br/>

```python
nums = [6, 1, 7, 23, 10, 24]
total = 0
for i in nums:
    total += i
print(total)
```
<br/><br/>



## 6. 링크 삽입
인라인 링크는 `[링크를 걸고 싶은 텍스트](링크 주소)`의 형태로 작성한다.

URL을 그대로 삽입하면 URL에 언더라인이 생기고 해당 페이지와 연결된다.

### **■ Markdown**  
```
URL 링크 : https://srcho01.github.io/

인라인 링크 : [나의 블로그](https://srcho01.github.io/)
```

### **■ Result** 
URL 링크 : https://srcho01.github.io/

인라인 링크 : [나의 블로그](https://srcho01.github.io/)<br/><br/><br/>



## 7. 이미지 삽입
`![이미지 대체 텍스트](이미지 주소)`의 형태로 작성한다.

이미지 주소로는 로컬 저장소의 이미지 경로나 이미지 링크를 삽입한다.

※ 로컬 저장소의 이미지 경로는 상대 경로로 작성한다.

### **■ Markdown**  
```
![GitHub](/assets/github.jpg)

![GitHub](https://i.imgur.com/Oyc3zPW.jpg)
```

### **■ Result**  
![GitHub](/assets/github.jpg)

![GitHub](https://i.imgur.com/Oyc3zPW.jpg)
